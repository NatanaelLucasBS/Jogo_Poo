# Projeto: Jogo POO

## Descrição
Este projeto foi desenvolvido como atividade avaliativa da disciplina **Programação Orientada a Objetos (TAD0009)**, do curso **Análise e Desenvolvimento de Sistemas - UFRN**.

O objetivo é aplicar os conceitos fundamentais de **Programação Orientada a Objetos (POO)** — como herança, abstração, encapsulamento, polimorfismo e interfaces — por meio da simulação de um jogo simples em que personagens jogáveis e não jogáveis interagem entre si através de ataques normais e especiais.
